# mkdocs 安装

## 创建虚拟环境

## 安装相应包

```bash
pip install mkdocs
pip install mkdocs-material
```

## 修改配置文件

使用主题 metrial，和语法高亮插件 codehilite。

```yml
site_name: Memo

theme:
  name: 'material'

markdown_extensions:
  - codehilite
```

## 常用命令

预览

```
mkdocs serve
```

生成站点

```
mkdocs build
```

## 部署脚本

```bash
# virtual env

git checkout gh-pages

deactivate

mkdocs build

git add *

git commit -m "update gh-pages"

git push origin gh-pages
```
